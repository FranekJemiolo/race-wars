Project Overview

Race Wars is a comprehensive trackday management system that brings together all aspects of motorsport event management into a unified platform. From traditional track days to custom "Need for Speed mode" races, the system provides real-time tracking, race control, and enforcement mechanics.

Key Features

  • Live GPS Tracking - Real-time position tracking with spatial analysis and performance metrics
  • Track Management - Comprehensive track database with geometry validation and spatial queries
  • Race Control - Advanced flag system, safety car management, and race state control
  • Custom Race Builder - Create custom routes with checkpoints and enforcement zones
  • Enforcement Layer - Speed violation detection and game mechanics
  • Real-time Updates - WebSocket-based live updates for all race events

Architecture

The system is built with a modern microservices architecture:

  • Database Layer - PostgreSQL with PostGIS for spatial data
  • Service Layer - Modular services for each domain
  • API Layer - RESTful APIs with authentication and authorization
  • Real-time Layer - WebSocket connections for live updates
  • Frontend - React-based SPA with real-time updates
