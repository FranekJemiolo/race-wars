Race Wars

Real-Time Multiplayer GPS Racing Engine

Race Anywhere, Anytime

A real-time multiplayer racing engine that transforms GPS data into competitive racing experiences. Create custom routes on real-world roads, track live positions, and compete with friends in closed-road racing events.

Perfect for trackdays, closed-road events, and organized racing meets. The system projects GPS positions onto predefined routes, validates checkpoints, and maintains live leaderboards with sub-second accuracy.

Live Map Visualization

🏁 Core Features

🗺️

Custom Route Builder

Draw race routes on OpenStreetMap or import GPX files. Define checkpoints, start/finish lines, and race rules with an intuitive admin console.

📍

GPS Projection Engine

Advanced geometry engine projects noisy GPS data onto route polylines with 5m accuracy using Turf.js and sophisticated smoothing algorithms.

🏁

Live Leaderboard

Real-time ranking system with sub-second updates. Supports multiple race types: sprint, checkpoint races, and loop-based lap racing.

🚨

Safety Awareness

Hazard zone system for safety warnings. Optional enforcement layers for controlled events. Route deviation detection and alerts.

📱

Mobile-First Design

Progressive Web App works on any device with GPS. Offline map caching for reliable operation in areas with poor connectivity.

⚡

Real-Time Sync

WebSocket-based architecture delivers position updates at 1-2Hz. Client-side interpolation ensures smooth 60fps rendering.

👥 Team-Based Racing

🏢

Team Management

Create, join, and manage racing teams with roles, permissions, and member statistics. Team-based competitions and leaderboards.

💬

Team Communication

Real-time team chat with reactions, system messages, and achievement notifications. Coordinate strategies and share race insights.

🏆

Team Competitions

Seasonal, tournament, and championship formats with team scoring. Multiple competition types and time-based rankings.

📊

Team Analytics

Performance metrics, achievements, and statistics. Track team progress, member contributions, and competitive insights.

🎬 Advanced Features

🎞️

Race Replay System

Video-like playback with analysis tools, lap times, sector analysis, and performance metrics. Review and learn from past races.

🏎️

Predefined Routes

Famous circuits: Monaco, Silverstone, Spa-Francorchamps, Suzuka, Nürburgring. Realistic track data and racing tips.

📢

Admin Event System

Real-time race management with event broadcasting, safety car deployment, weather warnings, and penalty systems.

🛡️

Anti-Cheat Detection

Advanced GPS validation and pattern analysis. Detect suspicious behavior and ensure fair competition.

Getting Started

1

Clone the Repository

git clone https://github.com/FranekJemiolo/race-wars.git

2

Install Dependencies

npm install (from root - installs all workspace packages)

3

Start the Server

npm run dev:server (runs on port 8080)

4

Start the Client

npm run dev:client (runs on port 5173)

5

Create a Race

Use the admin console to draw routes and configure race rules
